# User stories

- As a user I can **sign up** with email and set a PIN (password).
- As a user I can **sign in** with email and PIN (password).
- As a user I can **sign out**.
- As a user I can **log** weight and macros (carbs, protein, fat) for a day in one form; submitting again for the same day updates that dayâ€™s log.
- As a user I can **view history** of my logs in a date range (table).
- As a user I can **view and edit my profile** (e.g. display name).
- As a user I can **open the app on my iPhone** and log quickly (PWA, narrow viewport).
- As a user I can **toggle dark/light theme**; my preference is persisted (cookie).
