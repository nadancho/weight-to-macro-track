# 2026-03-28

## Woodland Cosmetics Platform

- Designed and implemented `packages/woodland/` — a portable badge/avatar/theme engine
- **Architecture:** Registry + Evaluator + Player State pattern (game-inspired, no classes)
  - Unified `Collectible` type — badges, avatar parts, and themes are all the same entity
  - `createRegistry()` factory, `evaluate()` + `resolveUnlocks()` pure functions
  - `PlayerStateAdapter` interface for host app storage
- **Theme system (Sprint 1, complete):**
  - `resolveTheme()` + `getUnlockedThemes()` with 14 unit tests
  - Cottagecore theme: forest greens, amber accents, warm cream text — derived from Gemini-generated woodland scene art
  - Classic Dark theme: preserves current app look as fallback
  - `WoodlandThemeProvider` applies CSS variables to `:root` inline styles
  - `ThemePicker` component on profile page with color swatches
  - Supabase migration: `profiles.theme` column + `user_collectibles` table
- **Nano Banana prompt:** Crafted and tested system instruction for Google AI Studio (Nano Banana Pro model) for generating pixel art sticker badges
  - Anthropomorphic animal cast: raccoon, fruit bat, ermine, hedgehog, otter family
  - Slice-of-life "Sylvanian Families meets pixel art" art direction
  - Magenta chroma key backgrounds for transparent sticker cutouts
  - Saved prompt as reusable template in AI Studio
- **Generated assets:** mama otter with kindergarten pups (Japanese uniforms), woodland foraging scene
- **Avatar system designed (not yet implemented):** 5-7 slot compositing, 12-frame sprite strips, hybrid body-animation + static-feature approach
- **Next:** Skinnable component system with 9-slice modular textures + animation triggers

## Key decisions
- Single `packages/woodland/` package, not three separate — unified Collectible abstraction
- Functional pattern (factory + pure functions), no classes — aligned with Zustand/Zod ecosystem
- Theme CSS variables applied as inline styles overlaying the existing `.dark {}` block
- Pre-generated art in-repo, not runtime Gemini calls

## Mobile UX Polish (Claude Code session)

- `log-cache-provider.tsx` replacing DataCacheProvider + cookie cache
